安装 dart：（使用apt-get安装就不用配置环境变量） 1.依次执行以下命令: $ sudo apt-get update $ sudo apt-get install apt-transport-https //科学上网 $ sudo sh -c 'wget -qO- https://dl-ssl.google.com/linux/linux_signing_key.pub | apt-key add -' $ sudo sh -c 'wget -qO- https://storage.googleapis.com/download.dartlang.org/linux/debian/dart_stable.list > /etc/apt/sources.list.d/dart_stable.list' 2.安装 Dart SDK: $ sudo apt-get update $ sudo apt-get install dart 3.输入 dart 检测是否安装完成。 Adds: //查看 dart 安装在何处 $ which dart //查看系统环境变量 $ echo $PATH Read more
